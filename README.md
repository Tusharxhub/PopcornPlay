## 🔐 API Configuration

To fetch movie data:

1. Sign up at [TMDB](https://www.themoviedb.org/)
2. Navigate to **Settings > API** and generate your API key.
3. Replace the placeholder in the code:

```javascript
const API_KEY = "YOUR_API_KEY_HERE";
```
